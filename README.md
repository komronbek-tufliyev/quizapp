# INSTALLATION

## 1. Clone this repository

```bash 
git clone https://github.com/komronbek-tufliyev/quizapp.git
```

### 2. Install dependencies

```bash
flutter pub get
```

### 3. Run the app

```bash
flutter run
```

Use the Firebase Console to create a project, then follow the steps below for your chosen platform:

### 2. configure app

